# Pfd
This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.18.

## Development server
To start the project:

1) Run ```npm install``` from the root directory
2) Run ```npm run start``` to start the project

## Building the design variables

1) change the design tokens in ```tokens.json```
2) From the root folder run ```npx style-dictionary build --config style-dictionary.config.js```
3) ```_variables.scss``` are generated
